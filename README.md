# Dam Anna Band Website

A 3D interactive website built with Three.js.

## About

This is the official website for Dam Anna. The site features a 3D guitar model rendered in the browser that rotates as you scroll through the page content.

## Technologies

- Three.js
- Vite
- JavaScript
- React
- HTML/CSS

## Future Plans
